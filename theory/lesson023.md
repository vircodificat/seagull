# Rule: `RuleColl`
When a word begins with "coll-", the outline begins with `K-`.
* [collect](KLECkT)
* [colleague](KOL/EG)
* [collar](KOL/AR)
* [collide]()
* [colony](KOL/ON)
* [color](KOL/OR)
* [column](KOL/OM)
* [collapse](KOL/AP)
* [collection]()
* [college](KOL/EJ)

Note StenEd has a variant where you can split words out when the stress is on the first syllable.

# Rule: `RuleCo`
When a word begins with "co-" where "co-" is used as a prefix, use `KO/`.
* [coordinate](KO/ORD/NAyT)
* [cohabitate](KO/HAB/TAyT)
* [coauthor]()
* [cooperate](KO/OP/RAyT)
* [coexist](KO/EG/ST)

# Rule: `RuleCol`
When a word begins with "col-" without a doubled "l", write it as `KOL/`.
* [colon](KOL/ON)
* [color](KOL/OR)
* [collapse](KOL/AP)
* [colleague](KOL/EG)
* [column](KOL/OM)
* [college](KOL/EJ)
* [collar](KOL/AR)

# Rule: `RuleCom`
When a word begins with "com-" without a doubled "m", write it as `KO*M/`.
* [complete](KO*M/PLEeT)
* [combine]()
* [combat](KO*M/BAT)
* [comfort](KO*M/FORT)
* [command](KO*M/MAND)
* [commence]()
* [commerce]()

# Rule: `RuleCorUnstressed`
When a word begins with "cor-" when the second syllable is accented, use `KR-`.
* [correct](KRECkT)
* [corrupt](KRU*PT)
* [corrode]()
* [corrosive]()

# Rule: `RuleCorStressed`
When a word begins with "cor-" when the first syllable is accented, use `KOR/` and drop a following weak vowel (if any).
* [correlate](KOR/LAyT)
* [corridor](KOR/DOR)
* [correspond](KOR/RES/POND)
* [corroborate](KOR/ROB/RAyT)
